 # Java Beginner Project

A basic calculator built using Java.

* Accepts user input for two numbers and the desired operation

## How It Works

1. Enter the first number.
2. Enter the second number.
3. Choose one of the four available operations:

4. The program displays the result.

## Getting the Code

To use this project, either:

* Clone the repository:

```bash
git clone https://github.com/beche-kis/beginner_projects
```

* Or download the source code and open `calculator.java` in your preferred Java IDE.

## Technologies Used

* Java
* Scanner Class for user input

> A beginner-friendly Java calculator that performs addition, subtraction, multiplication, and division based on user input.
